Sistema Gerenciador de Tarefas Online - Assistência Técnica

📌 Sobre o Projeto
Este projeto foi desenvolvido como parte da disciplina Projeto Integrador III-B do curso de Análise e Desenvolvimento de Sistemas da PUC Goiás. O sistema visa solucionar problemas de gestão em uma organização real por meio da extensão universitária.

O parceiro selecionado para este projeto foi uma MEI de Venda e Manutenção de Computadores, que apresentava dificuldades em organizar o fluxo de máquinas em bancada e cumprir prazos de entrega.

🛠️ Funcionalidades Obrigatórias (MVP)
O sistema contempla os requisitos mínimos estabelecidos no plano de ensino:

Cadastro e Autenticação: Acesso controlado para o gestor da MEI.
Gestão de Ordens de Serviço (Tarefas): Criação de demandas com definição de prazos e atribuição.
Quadro Kanban: Visualização do status das manutenções em colunas "A fazer", "Em andamento" e "Concluído".
Integração com Calendário: Monitoramento visual de prazos e entregas.
