# EA tech | Sistema de Gerenciamento para Assistência Técnica

Este projeto é um **Sistema Gerenciador de Tarefas Online** desenvolvido para MEIs de Manutenção de Computadores. Ele visa otimizar o fluxo de trabalho, desde a entrada do equipamento até a entrega final ao cliente.

📌 Este repositório faz parte da disciplina **Projeto Integrador III-B** do curso de Análise e Desenvolvimento de Sistemas da **PUC Goiás**.

---

## 🛠️ Tecnologias Utilizadas

O sistema foi construído utilizando uma stack moderna e eficiente para aplicações leves:

*   **Backend**: Node.js com Framework Express.
*   **Banco de Dados**: SQLite3 (Persistência local em arquivo `.db`).
*   **Frontend**: HTML5, Vanilla JavaScript e Tailwind CSS (Styling).
*   **Design**: Material Symbols (Ícones) e Google Fonts (Inter/Manrope).

---

## 🚀 Como Rodar o Projeto

Siga os passos abaixo para configurar e executar o sistema em sua máquina local:

### 1. Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado (versão 16 ou superior recomendada).

### 2. Clonar o Repositório
```bash
git clone https://github.com/GustavoL7/P.I.-III.git
cd P.I.-III
```

### 3. Instalar Dependências
No diretório raiz do projeto, execute:
```bash
npm install
```

### 4. Iniciar o Servidor
Para rodar a aplicação, utilize:
```bash
npm start
```
O servidor será iniciado em `http://localhost:3000`.

## ✨ Funcionalidades Principais

*   ✅ **Dashboard Dinâmico**: Visão geral de ordens totais, em reparo e concluídas.
*   ✅ **Gestão de Ordens (CRM)**: Cadastro completo de clientes, equipamentos e diagnósticos.
*   ✅ **Quadro Kanban**: Gerenciamento visual via Drag & Drop.
*   ✅ **Calendário de Prazos**: Acompanhamento mensal de datas de entrega.
*   ✅ **Segurança**: Autenticação de usuário e proteção de rotas privadas.
